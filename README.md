# Secure Chat Application

## Description
A secure multi-client chat application built using Python sockets, SSL/TLS encryption, SQLite database, and bcrypt authentication.

## Features
- Secure communication using SSL/TLS
- Multi-client support using threading
- User authentication (Register/Login)
- Password hashing using bcrypt
- Message storage using SQLite

## Technologies Used
- Python
- Socket Programming
- SSL/TLS
- SQLite3
- bcrypt

## How to Run

### Start Server

python server/server.py
